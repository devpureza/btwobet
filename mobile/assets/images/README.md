# Imagens do app

## `login_hero.png`

Ilustração exibida acima do formulário na tela de login.

- **Caminho:** `assets/images/login_hero.png`
- **Sugestão:** PNG ~800×400, fundo transparente, estilo cartoon festivo da Copa
- **Substituição:** basta salvar o arquivo neste diretório com o mesmo nome e rodar `flutter pub get` (se necessário) + hot restart

Se o arquivo não existir, o app usa um header temático com ícones (bola + troféu + “Copa 2026”).
